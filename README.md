# GooglePlaces.Xamarin
Lightweight C# wrapper for Google Places API.
## Installation
```csharp
// Install release version
Install-Package GooglePlaces.Xamarin

```

## Supported frameworks
* .NET 4.5+
* ASP.Net Core 1.0
* Xamarin Android
* Xamarin iOS 
* Xamarin iOS Classic
* Windows 8
